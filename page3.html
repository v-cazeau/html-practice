Week 1 - Day 1: Keyboard Shortcuts

Operating System

  • Command + C: Copy
  • Command + V: Paste
  • Command + Q: Quit
  • Command + W: Close Window
  • Command + Z: Undo
  • Command + S: Save

Visual Studio Code(VSC)

  • Command + B: Toggle Sidebar
  • Command + J: Toggle Terminal
  • Command + Z: Soft Wrap
